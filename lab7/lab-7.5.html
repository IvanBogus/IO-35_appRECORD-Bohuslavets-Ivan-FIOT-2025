ЗАВДАННЯ №4 — Генератор промісів (Promise + iziToast)

Суть: форма створює проміс з затримкою delay. Користувач обирає стан (fulfilled/rejected). Результат показується сповіщенням через iziToast.

Фрагмент коду:

function generateDelayedPromise(delay, state) {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      if (state === "fulfilled") resolve(delay);
      else reject(delay);
    }, delay);
  });
}

promiseFormElement.addEventListener("submit", (event) => {
  event.preventDefault();

  const delayValue = Number(promiseFormElement.elements.delay.value);
  const promiseState = promiseFormElement.elements.state.value;

  generateDelayedPromise(delayValue, promiseState)
    .then((ms) => {
      iziToast.success({
        title: "✅ Success",
        message: `Fulfilled promise in ${ms}ms`,
        position: "topRight"
      });
    })
    .catch((ms) => {
      iziToast.error({
        title: "❌ Error",
        message: `Rejected promise in ${ms}ms`,
        position: "topRight"
      });
    });
});

Перевірка: введи delay (наприклад 1000), обери fulfilled → submit → має бути success; обери rejected → submit → має бути error.
