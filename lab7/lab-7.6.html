ЗАВДАННЯ №5 — Пошук зображень (Pixabay API + SimpleLightbox)

Суть: реалізовано сторінку пошуку зображень по ключовому слову. Під час запиту показується loader, результати відображаються картками, а перегляд великого зображення — через SimpleLightbox. Повідомлення/помилки — через iziToast.

Фрагмент коду запиту:

const PIXABAY_API_KEY = "YOUR_PIXABAY_API_KEY";
const API_BASE_URL = "https://pixabay.com/api/";

function retrieveImages(query) {
  const params = new URLSearchParams({
    key: PIXABAY_API_KEY,
    q: query,
    image_type: "photo",
    orientation: "horizontal",
    safesearch: "true",
    per_page: 40
  });

  return fetch(`${API_BASE_URL}?${params.toString()}`).then((response) => {
    if (!response.ok) throw new Error(`HTTP error: ${response.status}`);
    return response.json();
  });
}

Фрагмент коду генерації розмітки:

function generateGalleryMarkup(images) {
  return images.map(({ webformatURL, largeImageURL, tags, likes, views, comments, downloads }) => `
    <li class="gallery__item">
      <a class="photo-card__thumb" href="${largeImageURL}">
        <img class="photo-card__image" src="${webformatURL}" alt="${tags}" loading="lazy" />
      </a>
      <div class="photo-card__info">...</div>
    </li>
  `).join("");
}

Перевірка: введи запит (наприклад cats) → Search → має зʼявитись список карток, при кліку відкривається lightbox. Якщо запит порожній або результатів 0 — зʼявляється сповіщення.

Примітка: для реальної роботи потрібен дійсний ключ Pixabay у змінній PIXABAY_API_KEY.
