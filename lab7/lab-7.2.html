ЗАВДАННЯ №1 — Кошик товарів та пагінація

Суть: реалізовано сторінку з каталогом товарів, розбитим на сторінки (pagination), та кошиком. Користувач може додавати товар у кошик, змінювати кількість (+/−) та очищувати кошик.

Ключові моменти реалізації:

  • каталог показується порціями по N елементів (розрахунок індексів за номером сторінки);
  • кошик зберігається як структура з ключем id товару та кількістю;
  • обробка кліків виконана через делегування подій.

Фрагмент JavaScript-коду:

const ITEMS_PER_PAGE = 6;
let activePage = 1;

const shoppingCart = {}; // { [id]: { product, quantity } }

function displayProducts() {
  const start = (activePage - 1) * ITEMS_PER_PAGE;
  const end = start + ITEMS_PER_PAGE;
  const pageItems = productInventory.slice(start, end);

  productGridContainer.innerHTML = pageItems.map(p => `
    <article class="product-card">
      <h3 class="product-card__title">${p.name}</h3>
      <p class="product-card__price">${p.price} ₴</p>
      <button class="btn btn--primary" data-add-id="${p.id}">Додати</button>
    </article>
  `).join("");
}

productGridContainer.addEventListener("click", (event) => {
  const btn = event.target.closest("button[data-add-id]");
  if (!btn) return;

  const id = Number(btn.dataset.addId);
  insertProductToCart(id);
});

Перевірка: додай кілька товарів у кошик, зміни кількість, перейди між сторінками та переконайся, що підрахунок суми/кількості коректний.
