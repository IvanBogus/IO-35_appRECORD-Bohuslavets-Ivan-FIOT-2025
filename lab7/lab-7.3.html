ЗАВДАННЯ №2 — Галерея + форма збереження стану (localStorage)

Частина 1: галерея зображень. Елементи створюються з масиву об’єктів, а при кліку відкривається модальне вікно (lightbox).

Фрагмент коду генерації галереї:

function buildGalleryHTML(items) {
  return items.map(({ preview, original, description }) => `
    <li class="gallery__item">
      <a class="gallery__link" href="${original}">
        <img class="gallery__image"
             src="${preview}"
             data-source="${original}"
             alt="${description}" />
      </a>
    </li>
  `).join("");
}

Частина 2: форма зворотного звʼязку. Стан полів зберігається у localStorage та відновлюється після перезавантаження сторінки.

Фрагмент коду localStorage:

const FORM_STATE_KEY = "feedback-form-state";
const formState = { email: "", message: "" };

feedbackFormElement.addEventListener("input", (event) => {
  const { name, value } = event.target;
  if (!(name in formState)) return;

  formState[name] = value.trim();
  localStorage.setItem(FORM_STATE_KEY, JSON.stringify(formState));
});

function loadFormState() {
  const saved = localStorage.getItem(FORM_STATE_KEY);
  if (!saved) return;

  const parsed = JSON.parse(saved);
  formState.email = parsed.email || "";
  formState.message = parsed.message || "";

  feedbackFormElement.elements.email.value = formState.email;
  feedbackFormElement.elements.message.value = formState.message;
}

Перевірка: введи email/message → перезавантаж сторінку → переконайся, що дані відновились. Після submit — сховище очищається.
