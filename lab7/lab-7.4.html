ЗАВДАННЯ №3 — Таймер зворотного відліку

Суть: користувач обирає дату/час через flatpickr. Якщо дата в минулому — показується помилка через iziToast. При натисканні Start запускається відлік та оновлення полів щосекунди.

Фрагмент коду (валідація дати):

flatpickr("#datetime-picker", {
  enableTime: true,
  time_24hr: true,
  defaultDate: new Date(),
  minuteIncrement: 1,
  onClose(selectedDates) {
    const selectedDate = selectedDates[0];

    if (!selectedDate || selectedDate <= new Date()) {
      iziToast.error({
        title: "Помилка",
        message: "Please choose a date in the future",
        position: "topRight"
      });
      startButtonElement.disabled = true;
      return;
    }

    chosenDateTime = selectedDate;
    startButtonElement.disabled = false;
  }
});

Фрагмент коду (конвертація ms → дні/год/хв/сек):

function parseMilliseconds(ms) {
  const SECOND_MS = 1000;
  const MINUTE_MS = SECOND_MS * 60;
  const HOUR_MS = MINUTE_MS * 60;
  const DAY_MS = HOUR_MS * 24;

  const days = Math.floor(ms / DAY_MS);
  const hours = Math.floor((ms % DAY_MS) / HOUR_MS);
  const minutes = Math.floor(((ms % DAY_MS) % HOUR_MS) / MINUTE_MS);
  const seconds = Math.floor((((ms % DAY_MS) % HOUR_MS) % MINUTE_MS) / SECOND_MS);

  return { days, hours, minutes, seconds };
}

Перевірка: вибери дату на 1–2 хв в майбутньому → Start → переконайся, що значення змінюються щосекунди.
