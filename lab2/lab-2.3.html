Приклад використання різних селекторів у моєму файлі css/style.css: 

/* селектор тегу */
header {
  background-color: #1b2a41;
  color: #ffffff;
}

/* селектор класу */
.hero__title {
  font-size: 40px;
  font-weight: 700;
}

/* селектор ідентифікатора */
#main-nav {
  font-size: 14px;
}

/* універсальний селектор */
*,
*::before,
*::after {
  box-sizing: border-box;
}

/* селектор атрибуту */
input[type="date"] {
  cursor: pointer;
}

/* дочірній селектор */
.contacts__address a {
  color: #e63946;
}

/* сусідній селектор */
.list li + li {
  margin-top: 6px;
}

У подальших підрозділах (селектори тегу, класу, ідентифікаторів, інші селектори) наведено окремі приклади з HTML-та CSS-кодом та скріншотами з браузера.
