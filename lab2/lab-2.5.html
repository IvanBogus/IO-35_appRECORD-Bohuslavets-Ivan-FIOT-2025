Застосування селектора класу у моєму веб-застосунку:

HTML-код елемента (карта номеру «Орія»):

<article class="room-card">
  <img src="public/Oriya.png" alt="Номер Орія" class="room-card__image" />
  <div class="room-card__body">
    <h3 class="room-card__title">Орія</h3>
    <p class="room-card__text">
      Ідеальне місце для романтичного відпочинку або розслаблюючого вікенду.
    </p>
    <a href="#booking" class="btn btn--ghost">Вибрати номер</a>
  </div>
</article>

CSS-код елемента (селектори класів):

.room-card {
  background-color: #ffffff;
  border-radius: 18px;
  overflow: hidden;
  box-shadow: 0 8px 20px rgba(15, 23, 42, 0.08);
  display: flex;
  flex-direction: column;
}

.room-card__title {
  font-size: 18px;
  font-weight: 600;
}

.room-card__text {
  font-size: 14px;
  color: #52606d;
}

Пояснення: селектори класів дозволяють повторно використовувати однакове оформлення для кількох карток номерів («Орія», «Луна», «Аврора», «Верона»). Це зручно для підтримки єдиного стилю та спрощує зміну дизайну.
