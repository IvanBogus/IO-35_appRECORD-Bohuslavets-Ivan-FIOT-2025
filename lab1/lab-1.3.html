2. Структура сторінки «about.html» (опис предметної галузі):

<!DOCTYPE html>
<html lang="uk">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Hotel Comfort — опис предметної галузі</title>
</head>
<body>
  <h1>Hotel Comfort — опис предметної галузі</h1>
  <p>Сайт готелю для ознайомлення з послугами, номерами, контактами та бронюванням.</p>

  <h2>Бізнес-логіка (скорочено)</h2>
  <ul>
    <li>Перегляд категорій номерів і тарифів.</li>
    <li>Пошук/бронювання за датами та кількістю гостей.</li>
    <li>Контакти й зворотний зв’язок.</li>
  </ul>

  <h2>Функціональні вимоги</h2>
  <ol>
    <li>Заголовки різних рівнів на головній сторінці.</li>
    <li>Зображення номерів та зон готелю.</li>
    <li>Таблиця тарифів.</li>
    <li>Списки (ul та ol).</li>
    <li>Форми пошуку номера та зворотного зв’язку.</li>
  </ol>

  <h2>Нефункціональні вимоги</h2>
  <ul>
    <li>HTML5, сучасні браузери.</li>
    <li>Репозиторій GitHub, публікація на GitHub Pages.</li>
  </ul>
</body>
</html>
