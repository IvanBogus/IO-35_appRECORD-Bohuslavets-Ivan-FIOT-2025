Тема предметної галузі: вебзастосунок готелю «Hotel Comfort».

Опис предметного середовища: вебсайт готелю призначений для ознайомлення гостей з типами номерів, умовами проживання, додатковими послугами (ресторан, SPA, галерея), а також для первинного бронювання номерів онлайн.

Бізнес-логіка (скорочено):

  • Перегляд категорій номерів, їх описів та тарифів.
  • Пошук / попереднє бронювання номера за датами й кількістю гостей.
  • Ознайомлення з послугами ресторану та SPA-зони.
  • Перегляд фотогалереї готелю.
  • Отримання контактної інформації та відправка повідомлення через форму зворотного зв’язку.

Функціональні вимоги:

  • Наявність заголовків різних рівнів на головній сторінці (h1–h3).
  • Використання зображень номерів та зон готелю.
  • Таблиця з тарифами номерів.
  • Списки (маркірований та нумерований) для переваг та кроків бронювання.
  • Форма пошуку/бронювання номера.
  • Форма зворотного зв’язку у блоці контактів.
  • Посилання між сторінками: головна сторінка готелю та сторінка опису предметної галузі.

Нефункціональні вимоги:

  • Технології: HTML5 без використання серверного бекенду.
  • Підтримка сучасних браузерів (Chrome, Firefox, Edge, Safari).
  • Робота у середовищі Visual Studio Code.
  • Система контролю версій Git, розміщення проєкту на GitHub.
  • Публікація живої сторінки за допомогою GitHub Pages.
